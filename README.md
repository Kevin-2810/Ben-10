# 🛸 OMNITRIX X – DNA Archive

An immersive futuristic web experience inspired by the legendary Omnitrix from Ben 10. Explore a digital DNA archive featuring interactive alien transformations, animated sci-fi visuals, holographic interfaces, and detailed alien profiles.

## 🚀 Features

- ⌚ Interactive Omnitrix Dial Interface
- 👽 10 Unique Alien DNA Samples
- 🎨 Futuristic Sci-Fi UI Design
- ⚡ Dynamic Alien Selection System
- 🌌 Animated Background Effects
- 🔥 Smooth Scroll & Transform Animations
- 📱 Fully Responsive Layout
- 💫 Neon Visual Effects & Holographic Styling
- 🎯 Interactive Transformation Sequence

## 👾 Alien Collection

- Heatblast
